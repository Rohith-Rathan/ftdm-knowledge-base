# Navigation List Template

## Screen Navigation Structure

### Main Navigation
1. **Login Screen** - `login_screen.png`
   - **Purpose**: User authentication
   - **Description**: Login form with email/password fields
   - **Assets**: login_icon.svg, background_image.png

2. **Dashboard Screen** - `dashboard_screen.png`
   - **Purpose**: Main dashboard with widgets
   - **Description**: Overview of user data and quick actions
   - **Assets**: dashboard_icons/, charts/, widgets/

3. **Profile Screen** - `profile_screen.png`
   - **Purpose**: User profile management
   - **Description**: View and edit user profile information
   - **Assets**: profile_avatar.png, edit_icon.svg

4. **Settings Screen** - `settings_screen.png`
   - **Purpose**: Application settings
   - **Description**: Configure application preferences
   - **Assets**: settings_icons/, toggle_components/

5. **Reports Screen** - `reports_screen.png`
   - **Purpose**: Data reports and analytics
   - **Description**: View reports and analytics data
   - **Assets**: chart_icons/, export_buttons/
